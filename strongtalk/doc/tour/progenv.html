The Programming Environment

To start with, if you plan on doing much browsing around, you might want to read the page on Keyboard Usage (then press"Back" to return to this page).

The Launcher

The launcher provides Transcript functionality, as in other Smalltalks. It is the first window opened, and usually ends up right behind the Start Page browser. The launcher toolbar provides quick access to some common functions:

  • : Open a browser on the Start Page.
  • : Find and open any top level construct by name (wildcards allowed).
  • : Find implementors of a message (wildcards allowed). Sorry about the lousy icon.
  • : Find senders of a message (wildcards allowed). Another lousy icon.
  • : Open a filtered hierarchy browser showing only the classes that users typically need.
  • : Open a hierarchy browser on the entire class hierarchy
  • : Open a workspace.
  • : Open a text editor.
  • : Browse the documentation.
